Clock Speed is the frequency that a processor runs at (ex. Hz, GHz) the higher the faster. the socket and chipset is the formations of pins on the bottom of the CPU and Motherboard. The two socket types are LGA and PGA.The architecture consist of 32-Bit, 64-bit, 86x=32-bit. The make and model is either a Intel or AMD and only works with motherboards built to use one of the two brands and only one of them. Host Bus Speed is the speed in which data is transferred between the processor and the chipset and the faster it is the better the performance. Cache Size is used to improve performance and can run at 128KB, 256KB,512KB, 1MB, or 2MB. Process Size is a process to reduce process size and use to come in 350nm, 250nm, and 180nm (nm = Nanometers). Newer ones use 250nm, 180nm, and 130nm. The goal is to get down to 65nm. SSE3 can make multimedia applications that support it run faster. 64 bit support was not really needed until windows vista and is back compatible with 32 bit software. Protected execution stops executable code from a non executable memory space. Power Reduction reduces the processor speed when idle or lightly loaded. Dual core support intel and amd put two cores to push the boundaries of what they could do.
