A Central Processing Unit (CPU) is a device inside the computer that can be most related to the brain. It takes data from many other parts like the RAM and “processes” it so that you can see it. An easy way of understanding how it works would be that it

  1. Takes complex info from other parts
  2. Makes it a not as complex.
  3. Completes the tasks were contained in the info.

Inside of the processor there are things called “cores”. The more cores it has the more it can process. Currently 2-core and 4-core processors are the most common to be in a pc but there are also 6-core processors that many people use.
