RAM stands for "Random Access Memory". The RAM acts as a short-term memory component. It stores data and machine codes. The most common RAM’s people buy are the DDR4, but for older systems they would buy DDR2 or DDR3. For video game space you would use VRAM which denotes the memory available to a graphics chip or built on to a graphics card. Some nice graphic cards utilize a unique form of RAM called High-Bandwidth-Memory. The RAM makes all electronic devices run correctly. It also allows data items to be read or written in almost the same amount of time irrespective. It is not designed to stay focus on one thing forever. It moves on as soon as it’s noticed. Any unsaved data will be deleted if you close a program without saving. Data from the RAM can be read from anywhere at nearly at the same speed. It is the most slowest component in a laptop.
