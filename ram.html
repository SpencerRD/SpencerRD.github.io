A RAM is a integrated circuit that stores information for imident use for a computer. The RAM acts as a short-term memory component. It stores data and machine codes. The most common RAM’s people buy are the DDR4, but for older systems they would buy DDR2 or DDR3. A DDR is a double rate dynamic random- access memory class of memory integrated circuits used in computers. The RAM makes all electronic devices run correctly. It also allows data items to be read or written in almost the same amount of time irrespective. It is not designed to stay focus on one thing forever. It moves on as soon as it’s noticed. Any unsaved data will be deleted if you close a program without saving. Data from the RAM can be read from anywhere at nearly at the same speed with that it makes it the slowest component in a computer.
